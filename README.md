# Inkwell - A Full-Stack Blogging Platform

Inkwell is a responsive and feature-rich full-stack blogging platform built using Python, Django, and PostgreSQL. It features a secure user authentication system that ensures isolated user access and management for blog posts.

## 🚀 Live Demo & Repository
- **Live Link:** [www.inkwell.pro.bd](https://www.inkwell.pro.bd)
- **GitHub Repository:** [https://github.com/atonu-a/inkwell](https://github.com/atonu-a/inkwell)

> ⚠️ **Note on Live Demo:** This application is hosted on Render's free tier. If the site takes around 40-50 seconds to load initially, it is completely normal as the free server goes to sleep after inactivity and needs a spin-up time to wake back up. Thank you for your patience!

---

## ✨ Features
- **User Authentication:** Secure registration, login, and logout systems ensuring private and isolated user environments.
- **Full CRUD Operations:** Authenticated users can smoothly Create, Read, Update, and Delete their own blog posts.
- **Relational Data Logic:** Robust backend structuring where articles, user profiles, and accounts are linked efficiently.
- **Responsive UI:** A clean, grid-based interface built with Bootstrap 5 that ensures flawless rendering across desktops, tablets, and smartphones.

---

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Database:** PostgreSQL (Hosted on Supabase)
- **Frontend:** Bootstrap 5, HTML5, CSS3
- **Deployment:** Render
